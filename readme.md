## About The Project

<img src="screenshot.png" >

<!-- GETTING STARTED -->
## Getting Started

### Prerequisites

* nodejs
* npm

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/lopezrunco/Mate-time-electron-js.git
   ```
2. Create "assets" folder and add two mp3 files with this names:
    ```sh
    rest-alarm.mp3 
    work-alarm.mp3
    ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Start the app
   ```sh
   npm start
   ```
